# Housing Price Prediction - Linear Regression

## Overview
This project implements a **linear regression model** to predict house prices based on various features from a housing dataset. It includes data preprocessing, model training, evaluation, and saving the trained model for future use.

**Repository Contents:**
- `task1_ml_linear_regression_kaggle.ipynb` – Jupyter notebook with full implementation
- `linear_regression_kaggle.pkl` – Saved trained model
- `housing.csv` – Dataset used
- `YashaswiniM_Documentation.pdf` – Project report
- `.gitignore` – Files to ignore in Git
- `README.md` – Project overview and instructions
